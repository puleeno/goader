# Danh sách host được hỗ trợ

Bảng dưới liệt kê các host mà Goader core hiện đã tích hợp sẵn. Danh sách được tổng hợp từ thư mục `src/Hosts/` trong mã nguồn.

| Host | PHP class | Ghi chú |
|------|-----------|---------|
| `bomtoon.com` | `Puleeno\Goader\Hosts\com\Bomtoon` | Webtoon Bomtoon. |
| `bookcube.com` | `Puleeno\Goader\Hosts\com\Bookcube` | Bookcube Comics. |
| `fanboylove.com` | `Puleeno\Goader\Hosts\com\Fanboylove` | Nội dung BL Fanboylove. |
| `iqiyi.com` | `Puleeno\Goader\Hosts\com\Iqiyi` | Host truyện Iqiyi. |
| `kanman.com` | `Puleeno\Goader\Hosts\com\Kanman` | Kanman/Kuaikan mobile. |
| `kuaikanmanhua.com` | `Puleeno\Goader\Hosts\com\Kuaikanmanhua` | Trang desktop. |
| `m.kuaikanmanhua.com` | `Puleeno\Goader\Hosts\com\Kuaikanmanhua\M` | Phiên bản mobile. |
| `lezhin.com` | `Puleeno\Goader\Hosts\com\Lezhin` | Hỗ trợ sử dụng cookie/token từ `config host`. |
| `comic.naver.com` | `Puleeno\Goader\Hosts\com\Naver\Comic` | Naver Webtoon desktop. |
| `m.comic.naver.com` | `Puleeno\Goader\Hosts\com\Naver\Comic\M` | Naver Webtoon mobile. |
| `ac.qq.com` | `Puleeno\Goader\Hosts\com\Qq\Ac` | Tencent AC (desktop). |
| `m.ac.qq.com` | `Puleeno\Goader\Hosts\com\Qq\Ac\M` | Tencent AC bản mobile. |
| `u17.com` | `Puleeno\Goader\Hosts\com\U17` | U17 Comics. |
| `text.io` | `Puleeno\Goader\Hosts\io\Text` | Trình tải nội dung văn bản. |
| `fshare.vn` | `Puleeno\Goader\Hosts\vn\Fshare` | Hỗ trợ tải từ Fshare (yêu cầu token/cookie tương ứng). |

> Nếu danh sách này chưa đầy đủ, bạn có thể bổ sung host mới và cập nhật lại tài liệu.

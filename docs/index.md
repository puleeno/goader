# Goader CLI

Goader là bộ công cụ dòng lệnh giúp tải nội dung từ nhiều host khác nhau, đồng thời cung cấp các tiện ích xử lý ảnh (đổi tên, tách layer, ghép ảnh). Bộ tài liệu này được xây dựng để hỗ trợ cả người dùng cuối lẫn developer muốn mở rộng hệ thống.

- **Sổ tay sử dụng:** Xem hướng dẫn chi tiết cho từng lệnh trong mục [Hướng dẫn sử dụng](README.md).
- **Dành cho developer:** Khám phá kiến trúc, hook và quy trình viết plugin ở mục [Hướng dẫn developer](DEVELOPERS.md).

> Bạn có thể dùng ô tìm kiếm (góc trên cùng bên phải) để tra nhanh theo từ khóa.
